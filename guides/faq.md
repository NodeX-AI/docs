# Часто задаваемые вопросы (FAQ)

  - **Почему модель может не распознать фотогграфию или вернуть ошибку?**

    ➤ В проекте NodeX.ai используются бесплтаные модели нейросетей, поэтому количество запросов в день для каждой ограничено. Распознавание фотографий может вернуть ошибку как раз по этой причине. Также при распознавании фотографий бот использует freeimage.host API, чтобы отправить фотографию на сервер и сделать из нее ссылку на изображение. По этой же причине, если freeimage.host API откажет, например, из-за технических работ, бот тоже на некоторое время перестанет распознавать изображения. Также бывают проблемы с провайдерами бесплатных моделей и модель, вместо ответа, может вернуть пустое сообщение, такое тоже бывает. Если такая ошибка случается с вами не единожды и слишком часто, пожалуйста, уведомите разработчика о ошибке по ссылке https://t.me/NodeX_project?direct.

  - **Почему модель называет себя другим именем/названием?**

    ➤ Модель может себя называть другим именем и это вполне нормально, так как этот проект использует бесплатные API, который предоставляет сервис openrouter.ai. Вероятно, провайдер API таким способом балансирует нагрузку между нагруженными нейросетями. Это не влияет на результат или работу бота, так как вы получите ответ на свой вопрос.

 - **Как удалить свои данные?**

    ➤ Все ваши данные (ID, история переписки) полностью удаляются через раздел "Опасная зона" в меню бота (/menu). Процесс необратим. Если вы удалили свой аккаунт и вам нужно воспользоваться функционалом бота, то вам необходимо снова зарегистрироваться при помощи команды /start. Если вы не зарегистрированы - вам недоступны все функции бота.

 - **Что означает ошибка API?**

   Если произошла ошибка API, бот отправляет следующее сообщение:

       "❌ Произошла ошибка API. Скорее всего, что-то произошло с API или количество запросов данной модели за день было исчерпано. Вы можете попытаться сменить модель. Пожалуйста, направьте эту ошибку разработчику."

    ➤ Ошибка API означает, что бот сформировал ваше сообщение в запрос для модели, но при во время оправки ответа модель, заместо ответа, направила ошибку. Дело в том, что API любой нейросетевой модели не всегда работает идеально: не серверах могут быть технические работы, или сама модель может просто просто так "упасть", обычно это исправляют за несколько часов. Также бот использует бесплатное API моделей и поэтому количество запросов в день ограничено, если количество запросов для модели закончилось, то она тоже возвращает ошибку. Если такая ошибка все же произошла, рекомендую вам сменить модель и пользоваться ботом дальше. На следующий день, вероятно, все исправится. Если эта ошибка вас преследует не единожды и слишком часто - вы можете ее разработчику.

  - **Что делаеть, если бот в овтет на сообщение ответил ошибкой?**

    ➤ Рекомендую вам отправить эту ошибку по ссылке, чтобы как можно скорее информировать пользователей насчет нее, или исправаить. Если это ошибка "Таймаут" - попробуйте отправить сообщение нейросети позже, возможно, модель перегружена.

  - **Какие модели ИИ доступны в боте?**

    ➤ Доступные модели искусственного интеллекта вы можете посмотреть в разделе меню "Доступные модели". Список доступных моделей может изменяться с течением времени.

  - **Чем NodeX.ai лучше других сервисов?**

    ➤ Главная фишка проекта — прозрачность и расширяемость. Вы не должны «верить на слово». Вы можете лично проверить код, который обрабатывает ваши сообщения. Кроме того, бот дает доступ к нескольким разным моделям в одном месте.

 - **Насколько безопасна моя переписка?**

    ➤ Все сообщения шифруются алгоритмом AES-256-GCM перед сохранением в базу данных. Ключ шифрования хранится отдельно. Исходный код открыт для проверки.

 - **Что означает AGPL-3.0?**

    ➤ Это лицензия открытого кода. Любой может проверить наш код, использовать его или создать свою версию бота, но обязан также открыть исходный код своих изменений. Это было сделано для безопасности пользователей и развития проекта.

 - **Вы несете ответственность за сгенерированный контент?**

    ➤ Нет. Весь контент создается ИИ автоматически. Вы используете его на свой страх и риск. Подробнее смотрите в Политике ограничения ответственности.

  - **Почему в правилах указана «нулевая ответственность»?**

    ➤ Потому что ИИ может генерировать неточный или нежелательный контент. Разработчик предоставляет инструмент, но не может контролировать, как он будет использован и какие решения вы примете на его основе. Это стандартная практика для подобных сервисов.

  - **Можно ли использовать контент от бота для коммерческих целей?**

    ➤ Да, вы можете использовать сгенерированные тексты и код по своему усмотрению. Однако, вы самостоятельно несете все риски и обязаны проверять качество, уникальность и правовой статус полученного контента перед использованием.

  - **Бот работает медленно. Это нормально?**

    ➤ Да, скорость ответа зависит от загрузки выбранной вами модели ИИ на стороне провайдера. Некоторые модели мощнее, но медленнее, другие — быстрее, но проще. Если проблема сохраняется, попробуйте сменить модель.

  - **Что делать, если бот не отвечает?**

    ➤ Сначала проверьте статус работы сервиса в нашем канале @NodeX_project. Если все работает, попробуйте отправить команду /start или перезапустить Telegram.
