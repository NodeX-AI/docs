# Возможности NodeX.ai

NodeX.ai сочетает в себе возможности современных нейросетей с максимальной простотой использования через Telegram.

## **Мультимодельный доступ**

Единый интерфейс для лучших языковых моделей:
- **Grok 4 fast** — высокоскоростная модель от xAI
- **Gemma 3** — сбалансированная производительность
- **DeepSeek R1T2 Chimera** — экспериментальная архитектура Chimera
- **Nemotron Nano 9B V2** — технология от NVIDIA

## **Безопасность и конфиденциальность**

### Шифрование
- Все сообщения шифруются алгоритмом **AES-256-GCM**
- Ключи шифрования хранятся отдельно от данных
- Никакие третьи стороны не имеют доступа к вашим диалогам

### Контроль данных
- Полное удаление аккаунта через меню "Опасная зона"
- Возможность удалить историю чата с нейросетью

## **Производительность**
- Оптимизированная архитектура для работы с множеством моделей
- Среднее время ответа в незагруженное время: 10-15 секунд

## **Технологический стек**

- **Python 3.12.3** — основной язык разработки
- **Aiogram** — взаимодействие с Telegram API
- **Aiohttp** — асинхронные HTTP-запросы
- **PostgreSQL** — хранение данных
- **Redis** — ограничение спама
