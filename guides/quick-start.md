# Быстрый старт

**Начать использование NodeX.ai можно всего за 3 простых шага.** Вам даже не нужно регистрироваться — достаточно аккаунта в Telegram.

## Для пользователей
1. **Запустите бота:**

    Перейдите в бота [@NodeX_aibot](https://t.me/NodeX_aibot) и введите команду /start

    **[!] Важно: при запуске бота при помощи команды /start, вы сразу соглашаетесь с Политикой конфиденциальности, Политикой ограничения ответственности и Пользовательским соглашением**

2. **Настройте бота, используя команду /menu (необязательно):**
    
    Бот предлагает множество различных настроек и управление аккаунтом. При помощи этой команды вы получите доступ к очистке истории сообщений, удалению аккаунта, изменению модели. Также оттуда вы можете узнать больше о проекте.

3. **Начните общение:**

    Просто напишите ваш вопрос или задачу в чат. Например:

    > «Напиши код простого калькулятора на языке программирования Python»

    > «Объясни квантовую физику простыми словами»

    > «Помоги составить бизнес-план»

## Для разработчиков (пример самостоятельного развертывания на Ubuntu Server)
**Требования: python 3.12.12 или python 3.12.3**
1. **Клонируйте репозиторий и установите все необходимые зависимости в директорию .venv**
```
    git clone https://github.com/NodeX-AI/NodeX.ai
    cd NodeX.ai
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
```
2. **Установите параметры в файле .env**
```
    touch .env
    nano .env
```

Файл .env должен выглядеть следующим образом:
    
```
BOT_API_TOKEN = <>

PG_URL = <>
REDIS_URL = <>

GEMMA3 = <>
DEEPSEEK = <>
MINIMAX = <>
NEMOTRON = <>
GROK4_FAST = <>
GROK4_FAST_ID = <>

GEMMA3_IMAGES = <>

IMAGE_HOSTING_KEY = <>
IMAGE_HOSTING_URL = <>

GEMMA3_BASE_URL = <>
DEEPSEEK_R1T2_CHIMERA_BASE_URL = <>
MINIMAX_M2_BASE_URL = <>
NEMOTRON_BASE_URL = <> 
GROK4_FAST_URL = <> 
AES_KEY = <>
```

Сгенерировать AES-ключ для шифрования вы можете при помощи команды:
    
    openssl rand -base64 32

3. **Запустите бота**

    Запустить бота вы можете при помощи команды

        python3 bot.py
